Standalone web applications and mini custom experimental frameworks

Two of my projects are standalone web applications, this I can present them right here, linking as live sites:

My own web application framework,
of course a very small and incomplete one, demonstrated on the example of a CRUD application. The site also presents its own source code in a self-documenting style.
A web gallery,
based on the web framework above. Most interesting capability of this gallery is graceful degradation through unobtrusive JavaScript that extends a solid core functionality: a nice fallback system turning JavaScripted widgets into simple page-reloding links with the right substitute actions. Scripting and executables

I have several minor supplementary projects, most of them are Linux shell programming scripts, or large interesting MySQL scriptfiles fulfilling surprizingly complex tasks with single query with pure declarative SQL (via mutliple subqueries nd large JOINs).

Due to their connections to proprietary projects, most of them are not publicable. Some minor public ones are:

  • A stopwatch with alarm clock - written as a bash-script
  • compact-pivotizer: a large AWK-script for extracting statistical probability deductions from large databases and visualizing dependencies in large pivot tables and graphs.
  • .po-replacer: a Haskell executable for converting tabular textfiles into po-files for internalization and localization.
